Get Tickets by Stack

Returns all tickets that are a part of the specified stack via JSON.

Parameter Name Variable Type Param Description Example Value
Stack Name String The Name of the Stack to be accessed. "Networking", "End-User"
Session Key String The User's session key that uniquely identifies them. "68gt6gr4e6d4e65"

Get Tickets by User Owned

Returns all tickets that are owned by a specified user (Techs and Users) via JSON.

Parameter Name Variable Type Param Description Example Value
Session Key String The User's session key that uniquely identifies them. "68gt6gr4e6d4e65"
User Type Integer The type of user 1, 2 or 3

POST Update ticket by Ticket ID

Updates the information of a ticket.

In addition to these values, all of the properties of the ticket will be passed with the updated information. This route will allow new comments to be added as well.

Properties of a ticket include the ticket's stack, subject, description, requestor, employee owned, and comments.

Parameter Name Variable Type Param Description Example Value
Session Key String The User's session key that uniquely identifies them. "68gt6gr4e6d4e65"
Ticket ID Integer The ticket's ID 14312

POST Create a ticket

Allows a ticket to be created by a user.

Parameter Name Variable Type Param Description Example Value

Subject String Short description of the issue. "Broken computer."
Description String Detailed description of the issue. "I touched my computer and it turned into a fireball. Please help me!!"
